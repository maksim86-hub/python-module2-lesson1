В языке программирования Python часто возникает необходимость преобразовать целое число в строку. Это может понадобиться, например, для объединения чисел с текстом или для их отображения в пользовательском интерфейсе. Для таких случаев Python предоставляет встроенную функцию `str()`.

### 1. Что такое функция `str()`?

`str()` — это встроенная функция Python, которая принимает на вход любой объект (в том числе и целое число) и возвращает его строковое представление. Это очень удобный способ преобразования чисел в строки, поскольку функция работает с любыми типами данных, включая целые числа (`int`), вещественные числа (`float`) и другие.

### 2. Пример использования `str()` с целыми числами

Рассмотрим простой пример, где мы преобразуем целое число в строку:

```python
number = 42
number_str = str(number)
print(number_str)  # Вывод: '42'
print(type(number_str))  # Вывод: <class 'str'>
```

В этом примере переменная `number` хранит целое число `42`. С помощью функции `str()` мы преобразуем его в строку и сохраняем результат в переменной `number_str`. После преобразования, тип данных переменной `number_str` будет `str`, то есть строка.

### 3. Где это может быть полезно?

**Объединение строк и чисел**: Часто требуется объединить текст с числовыми значениями, например, для вывода сообщений пользователю:

   ```python
   age = 25
   message = "Ваш возраст: " + str(age)
   print(message)  # Вывод: 'Ваш возраст: 25'
   ```

   В данном случае без преобразования целого числа в строку возникла бы ошибка, так как Python не позволяет объединять строки и числа напрямую.

### 4. Выполните следующий код:
```python
age = 25  # Инициализируем переменную с возрастом
message = "Ваш возраст: " + str(age)  # Преобразуем age в строку и объединяем с текстом
print(message)  # Вывод: 'Ваш возраст: 25'
```
